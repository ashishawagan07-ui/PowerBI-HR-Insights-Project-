# 🚀 HR Analytics Dashboard – Power BI Project 📊

## 📌 Project Overview

This project presents an interactive **HR Analytics Dashboard** built using **Power BI** to analyze employee data, workforce performance, attrition, attendance, and key HR metrics.

## 🛠️ Tools Used

* Power BI 📊
* Excel / SQL 🗂️
* Data Visualization 📈

## 📂 Dataset Information

The dataset contains employee details, attendance records, performance metrics, and HR-related KPIs.

## ✨ Key Features

✔️ Employee Performance Analysis
✔️ Attrition & Retention Tracking
✔️ Attendance Analysis
✔️ HR KPI Monitoring
✔️ Interactive Dashboard Visualizations
